Compound multiplication assignment operator for integral types.

Template Parameters

Parameters
scalarThe scalar value to multiply by.
Returns
Reference to the modified Vec2 object.

Multiplies each component of the vector by the integral scalar value. This specialized version uses Fxp's optimized integral multiplication for better performance on Saturn hardware.

Example usage:

Vector2D v(1, 2);
v *= 2; // Results in v = (2, 4) with optimized integral multiplication
