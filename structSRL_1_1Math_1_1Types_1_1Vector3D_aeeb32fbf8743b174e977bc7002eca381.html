template<Precision P = Precision::Default>
Fxp SRL::Math::Types::Vector3D::DistanceTo ( const Vector3D & other) const
inlineconstexpr

Calculate the Euclidean distance from this vector to another vector.

Template Parameters
PPrecision level for calculation
Parameters
otherThe other vector to calculate the distance to.
Returns
The distance as an Fxp value.

Computes the distance using the formula: sqrt((X - other.X)^2 + (Y - other.Y)^2 + (Z - other.Z)^2). The precision template parameter controls the calculation method:

  • Standard precision: Uses exact square root calculation
  • Turbo precision: Uses fast approximation

Example usage:

Vector3D v1(1, 2, 3);
Vector3D v2(4, 6, 8);
Fxp distance = v1.DistanceTo(v2); // Computes distance between the two points
Fxp fastDistance = v1.DistanceTo<Precision::Turbo>(v2); // Computes approximate distance (faster)
